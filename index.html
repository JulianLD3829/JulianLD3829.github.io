Applications You Can Get

To install an app, simply click the yellow button to add it, that will remove if it is already installed.

Note: the refresh/close policy still applies to Apps!

If you want to create apps, please fill out the form below. If you already have an app, please press "Developer Portal" below.

For how to develop apps, please open Dev Help.

Dev Help contains helpful snippets of code from DeskUtility to help you out, as well as tutorials!

Before using the Developer Portal:

1. Do you have your Developer ID?

2. Do you have your App ID? (Updates only.) You can find it in Line 2 of your code if you have already developed an app and were sent the revised version.

3. Have you been accepted in the Developer Form? (Most Important.) If you don't have an approved Developer Form, you cannot do ANYTHING here. Any submitted apps from here will be deleted if you are not approved.

4. Do you have your Developer Username? If you forgot it or would like to change it, please use your Developer ID.

If you have all of the above, you are in the clear.

App Developer Help

If you are here, thank you for shaping DeskUtility's future.

Devoloping a DeskUtility app is as simple as 1, 2, 3!

1. Fill out the form, and download DeskUtility code from the GitHub link. (Not required.)

2. If you get accepted, develop your app in a text editor and put your email and name in the code by doing this at the beginning of your code: . If it is not there, the software will be rejected with: "Invalid info."

3. Upload it or send it off as an update in the Developer Panel. Now you are done! Make sure to update to make sure your users don't get bored of it.

Any software with viruses WILL be removed and the creator of it WILL be BANNED from using ANY software creation services in DeskUtility.
